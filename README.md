# Micro-Investment Platform (Spare Change Investing)

## Introduction

This project helps users save money automatically from their daily expenses.
Whenever the user spends money, the system rounds the amount and saves the extra money as spare change.
When the saved amount reaches ₹100, it is automatically moved to investments.

Example:
If the user spends ₹87,

* Rounded amount = ₹100
* Spare amount = ₹13
* ₹13 is added to the wallet

---

## Problem

Many people spend small amounts every day but do not save them.
Those small amounts become a large loss over time.
Most investment apps need manual effort and larger amounts.

---

## Solution

Our platform automatically collects spare change and invests it.
This makes saving simple and easy for beginners.

---

## Main Features

* User login with mock OTP
* Aadhaar and PAN input
* Aadhaar and PAN masking
* Add daily expense amount
* Automatic spare change calculation
* Wallet to store spare money
* Auto investment when wallet reaches ₹100
* Dashboard to show savings and investments
* Goal-based saving
* Future prediction for 6 months and 1 year
* Streaks and badges
* Safety mode for high spending

---

## Execution Flow

1. User logs in.
2. User enters expense amount.
3. System rounds the amount.
4. Spare amount is calculated.
5. Spare amount is added to wallet.
6. If wallet reaches ₹100, it is automatically invested.
7. Dashboard updates automatically.
